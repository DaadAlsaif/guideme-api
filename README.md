# guideme-api

 guideme-api

 ##Features
 
for user(tourist)

 1.Login 

 2.Choose City to visit

 3.View all Tour Guides

 4.Search Tour Guides by .

 5.Reserve tour guide

 6.Update profile.

 7.Contact tour guide

 8.Cancel reservation


 ##Stack

 Runtime Environment: `Node.js`

 Language: `JavaScript`

 Web Server:` Fastify`

 ORM: `Prisma`

 DB: `Postgres`
